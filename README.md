### 예약 + 대기열 프로젝트 Front
---
사용자는 홈 화면에서 공연장을 선택한 뒤, 해당 공연장에 등록된 공연 목록에서 원하는 공연을 선택하여 대기열에 진입합니다.

대기열에서 순서가 확정되면 해당 공연의 회차(스케줄)를 선택할 수 있는 페이지로 이동하며, 이때부터 10분의 제한 시간이 주어집니다. 회차를 선택하면 좌석 선택 페이지로 이동하여 원하는 좌석과 인원 수를 지정할 수 있습니다.

좌석 선택 후 결제 페이지에서 최종 금액을 확인하고, 보유 중인 리워드 포인트를 사용하여 할인을 적용할 수 있습니다. 결제가 완료되면 예약이 확정되며, 홈 화면의 예약 내역 페이지에서 예약 정보를 확인할 수 있습니다.

대기열 확정 이후 10분의 제한 시간 내에 결제를 완료하지 않으면 대기열에서 자동으로 제거되어 홈 화면으로 이동됩니다. 또한, 예약 흐름 도중 다른 페이지로 이동하거나 브라우저 탭을 닫는 등 흐름을 이탈하는 경우에도 대기열에서 자동으로 제거됩니다.<br><br>
